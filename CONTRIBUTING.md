# Contributing Guidelines

Thank you for your interest in contributing to the Simple Interest Calculator project! We welcome contributions from developers of all skill levels to help empower community micro-finance tools.

## How to Contribute

### 1. Reporting Bugs
* Check the **Issues** tab to see if the bug has already been reported.
* If not, open a new Issue with a clear title, steps to reproduce, and expected behavior.

### 2. Suggesting Enhancements
* Open an Issue detailing the feature or enhancement you would like to see added.

### 3. Submitting Code Changes
1. **Fork** the repository to your own GitHub account.
2. **Clone** your forked repository locally:
   ```bash
   git clone [https://github.com/Mehransangi/simple-interest-calculator.git](https://github.com/Mehransangi/simple-interest-calculator.git)
